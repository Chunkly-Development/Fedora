## [v1.0.0-STABLE]

### Added
- Maintenance mode: toggle the server offline with a custom kick message and player whitelist.
- Customizable server MOTD with two lines and an optional countdown.
- Countdown system: set a target date and show the remaining time on the MOTD.
- Broadcast command to send messages to the whole network.
- Permission-based access for all commands.
- Full color code support (`&0`-`&f`) in messages and configs.
- Configuration files that reload automatically.

### Notes
- All permissions are listed in `permissions.txt`.
- Check `config.yml` to customize maintenance, MOTD, and countdown.
